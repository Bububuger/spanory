# Plan (2026-03-06) — npm 包名订正为 @spanory/spanory

## 背景
用户要求对外产出物名称从 `@spanory/cli` 订正为 `@spanory/spanory`，并保持 release/CI/文档一致。

## 目标
- npm 包名改为 `@spanory/spanory`。
- 所有活跃构建与发布脚本使用新 workspace 名。
- README（中/英）与关键文档改成新安装命令。
- lockfile 与最小验收通过，避免后续 CI 回归。

## 变更范围
- 包定义：`packages/cli/package.json`
- 构建发布：`.github/workflows/*.yml`、`scripts/release/*`、根 `package.json`
- 文档：`README.md`、`docs/README_zh.md`、`docs/langfuse-parity.md`、`docs/standards/test-baseline-spec.md`、`CHANGELOG.md`

## 验收标准
1. `rg -n "@spanory/cli"` 在业务文件中无残留（允许出现在本计划/归档说明中）。
2. `npm run --workspace @spanory/spanory build` 成功。
3. `npm run --workspace @spanory/spanory test:bdd` 成功。
4. `npm pack --workspace @spanory/spanory --dry-run` 显示包名为 `@spanory/spanory`。
