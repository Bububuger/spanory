# OpenCode Plugin 接入计划（Spanory）

## Goal
在现有 `claude-code` / `openclaw` 之外，新增 **OpenCode plugin 实时上报链路**，使 OpenCode 会话可稳定上报到 Langfuse OTLP 端点。

## Scope
- In scope:
  - 新增 `packages/opencode-plugin`（OpenCode 插件入口与运行时）
  - `spanory runtime opencode plugin install|uninstall|doctor` 管理命令
  - OpenCode 会话消息 -> Spanory canonical events 映射（复用 shared normalize）
  - 插件发送失败时本地 spool + retry（最小耐久）
  - 单元测试 + BDD 覆盖插件运行时与 CLI 管理命令
  - README / capability / parity / roadmap 更新
- Out of scope:
  - `runtime opencode hook/backfill` 通用 CLI 补数命令
  - OpenCode SSE 监听链路
  - LangSmith backend
